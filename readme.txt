安装条件
Apache + php + mysql/Sqlite3
Apache 需要安装并开启 url_rewrite (用于伪静态) 模块
最好支持目录配置  .htaccess 
如果不支持，请参考 local.htaccess 自行进行 apache 配置

安装方式
1. 解压 Alpaca 到网站目录
2. 确认  app 、 app/v/data 、 upfile 目录可写
3. 通过浏览器访问 alpaca 地址，自动进入安装程序
